🚫 Popup Blocked

Your browser blocked the Google sign-in popup.

How to Allow Popups:

  • Chrome/Edge: Click the popup icon 🚫 in the address bar (right side), then "Always allow popups from localhost"
  • Safari: Safari menu → Settings → Websites → Pop-up Windows → Allow for "localhost"
  • Firefox: Click the shield icon 🛡️ in address bar → Turn off blocking

After allowing popups, click "Retry Connection" below.

  1. Go to: Google Cloud Console - Credentials
  2. Find your OAuth 2.0 Client ID (NOT the API Key) and click the edit icon (pencil)
  3. Under "Authorized JavaScript origins", click "+ ADD URI"
  4. Add this exact origin (without /* at the end):
  5. Click "SAVE"
  6. Wait 1-2 minutes for changes to propagate, then refresh this page
Note: This is different from the API Key restriction. You need BOTH configured:
  • OAuth Client ID → Authorized JavaScript origins (no /*)
  • API Key → HTTP referrers (with /*)

After adding the origin, click "Retry Connection" below.
